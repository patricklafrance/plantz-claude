I want you to create a collection of agent skills to will map different profile of the Software Development Lifecycle. The goal is to build a collection of agent skills that will be used by an orchestor agent and different subagents to develop a feature.

The agent skills to create are:

- plantz-orchestrator: An agent skill for the orchestrator agent that will lead and be accountable for the development of the feature. The orchestrator agent will orchestrator the different steps of the process and coordinate subagents.
- plantz-plan: An agent skill to plan the development of the code of the feature. The planning subagents will elaborate the plan the output it in a file.
- plantz-code: An agent skill to execute the plan. The coding subagents will load the plan file and execute the plan.
- plantz-validator: An agent skill to validate the code generated by the coding subagents. With the help of the orchestrator agent, the validator subagents will

new skill named `plantz-code`. This skill must be loaded everytime an agent wants to write code. The skill will include the instructions for every phase of writing code. The main agent using will act as an orchetrator (and will be referenced going forward as the "orchestrator" agent) and will spawn subagents for different steps.
