I want you to create a collection of agent skills to will map different profile of the Software Development Lifecycle. The goal is to build a collection of agent skills that will be used by an orchestor agent and different subagents to develop a feature.

All agent skills must be created using the `skill-creator` agent skill and 2 subagents to assist and challenge each others.

The agent skills to create are:

- `plantz-sdlc-start`: An agent skill for the orchestrator agent that will lead and be accountable for the development of the feature. The orchestrator agent will orchestrator the different steps of the process and coordinate subagents.
- `plantz-sdlc-plan`: An agent skill to plan the development of the code of the feature. The planning subagents will elaborate the plan the output it in a file.
- `plantz-sdlc-code`: An agent skill to execute the plan. The coding subagents will load the plan file and execute the plan.
- `plantz-sdlc-validate`: An agent skill to validate the code generated by the coding subagents. With the help of the orchestrator agent, the validator subagents will look for issues in the code and output their findings in a file.
- `plantz-sdlc-document`: An agent skill to update the repository agent documentation. Documentarian subagents will look at the plan and the changes and identify drifts in the existing documentation. If any drifts are found, they will fix them.
